Design Rules

The constraints that keep it consistent.

01 — One Accent
Gold is the only accent color. Buttons, active states, highlights, badges — all gold. No rainbow categories.
02 — Bold, Not Decorative
Space Grotesk for headlines — geometric, modern, confident. Inter for everything else. No serifs, no scripts, no frills.
03 — Content Brings Color
User photos and activity images are where vibrancy lives. The UI chrome stays neutral and lets content pop.
04 — Functional Color Only
Coral = errors and destructive actions. Sage = success and confirmations. Never used decoratively.
05 — Breathe
Space is confidence. Generous padding, clean gaps, no cramming. If a screen feels busy, remove — don't rearrange.
