Design Rules

The constraints that create the luxury.

Rule 1
Gold is the only accent. Buttons, active states, highlights — all gold. No rainbow categories.
Rule 2
Coral is functional only. Errors, destructive actions, unread badges. Never decorative.
Rule 3
Content brings the color. User photos and imagery — that's where vibrancy lives. The UI itself stays neutral.
Rule 4
Headlines are Cormorant Garamond. Body is Inter. No exceptions. No third font.
Rule 5
Space is luxury. When in doubt, add padding. Nothing should feel cramped.
